# qiuUGM

 为一个叫做叶秋的虚拟主播编写的群管插件（符合真寻 BOT 插件标准）

> [!NOTE]
> 插件目前仍在开发中，请勿用于生产环境


## 特性

- [x] 群管群 — 所有命令仅在群管群内生效，群员违规将转发至群管群
- [x] 多群联合封禁 — 被封禁者禁止入群
  - `/封禁 <QQ>` — 封禁
  - `/解封 <QQ>` — 解除封禁
- [x] 累计警告 — 受到警告将禁言`次数*5`分钟
  - `/警告 <QQ> <次数>` — 修改警告次数
- [x] 记录查询 — 查询封禁状态、警告次数
  - `/查 <QQ>` — 查询对应号码记录
- [x] 色图识别 — 自动撤回+警告
- [ ] 图片 OCR — 自动撤回+警告含有黑名单词汇的图片
- [ ] 黑名单关键词 — 自动撤回+警告含有黑名单词汇的图片
